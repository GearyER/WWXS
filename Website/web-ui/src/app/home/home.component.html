Welcome to the Water Weather Stations Project!

The Water Weather Stations project is an endeavor by WWU students to create a buoy for documenting weather data in coastal locations. The Water Weather Stations project is composed of a mobile app, the physical buoy itself, and the website.

The App

The mobile application's primary function is the collection of data from the buoy itself. Connecting via bluetooth, the app can collect data from buoys, display this data on a mobile phone, and further make this data available to the website through the API.

The Buoy

The buoy is central to the Water Weather Stations project, containing an array of sensors that collect metrics such as salinity, turibidity, temperature, insolation and dissolved oxygen.

The Website

The Water Weather Stations website provides access to data collected from the deployed buoys, this data is provided to the website by the mobile application.
